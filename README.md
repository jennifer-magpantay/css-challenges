# css-challenges

## CSS 100 Days Challenge 

Matthias Martin created [100 days of CSS challenges](https://100dayscss.com/). They are all there for you to see, including entries from other people â€” but the point is for you to give it a shot yourself, of course.

To get into is simple:
1) Open the template
2) Recreate the given example
3) Post your code

Important: you have an timer running out!

### Goals

Replicate the challenges and learn how to do amazing stuff with pure CSS.
